ABOUT

Recursive Eclipse is a creative group of students who have set forth to create a web based application designed to make reserving rooms in the Ottenheimer Library easier and faster than ever.

Web Development

We implemented the use of HTML, CSS, JavaScript, jQuery, AJAX, JSON through Cloud 9 Workspace with a Bootstrap framework.

Server

We used PHP and Apache server to connect the server-side to the client-side.

Database

For the Database, we implemented the use of PHP MyAdmin server-side to connect MySQL to the client-side.
